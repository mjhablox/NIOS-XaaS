h2. Objective

Run the DDIaaS lease4 count scripts on *production* to get DHCP lease counts
for NA (52 accounts) and EU (20 accounts) customer accounts. This data is
needed to understand DDIaaS lease utilization across regions.

h2. Background

Two scripts are available for counting DDIaaS leases:

* *lease4_count.sh* — queries the cloud dhcp-leases DB (host_leases table).
  Supports {{--accounts-file}} to filter by account IDs from NA.txt / EU.txt.
  Supports {{--xaas}} mode for DDIaaS-only lease counts. This mode also queries
  the DDIaaS endpoint-manager DB to map endpoints to accounts.

Both scripts are located in:
{{Automation/CountInformation/}}

h2. Execution Steps

h3. Step 1: Teleport Login

{code:bash}
# Cloud control-plane clusters
tsh kube login us-com-1
# tsh kube login eu-com-1  (confirm EU cloud cluster name)

# DDIaaS clusters (needed for --xaas mode endpoint-manager query)
tsh kube login ddi-prd-nva1
tsh kube login ddi-prd-frk1
{code}

h3. Step 2: Run lease4_count.sh for NA

{code:bash}
cd Automation/CountInformation

# Standard report (all BloxOne leases from cloud dhcp-leases DB, filtered to 52 NA accounts)
bash lease4_count.sh --accounts-file NA.txt --limit 30 \
  --context teleport.services.sdp.infoblox.com-us-com-1 \
  2>&1 | tee NA_lease4_output.txt

# XaaS-only report (DDIaaS leases only, uses endpoint-manager for mapping)
bash lease4_count.sh --xaas --accounts-file NA.txt --limit 30 \
  --context teleport.services.sdp.infoblox.com-us-com-1 \
  --ddiaas-context teleport.services.sdp.infoblox.com-ddi-prd-nva1 \
  2>&1 | tee NA_xaas_lease4_output.txt
{code}

h3. Step 3: Run lease4_count.sh for EU

{code:bash}
# Standard report (filtered to 20 EU accounts)
bash lease4_count.sh --accounts-file EU.txt --limit 30 \
  --context <eu-cloud-context> \
  2>&1 | tee EU_lease4_output.txt

# XaaS-only report
bash lease4_count.sh --xaas --accounts-file EU.txt --limit 30 \
  --context <eu-cloud-context> \
  --ddiaas-context teleport.services.sdp.infoblox.com-ddi-prd-frk1 \
  2>&1 | tee EU_xaas_lease4_output.txt
{code}
